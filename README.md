<h1>Installation</h1>
<pre>$ git clone https://github.com/Ansup17174/randomproject.git</pre>
<pre>$ cd randomproject/randomproject</pre>
<pre>$ python -m venv venv</pre>
Activate virtual env on Windows:
<pre>/> ./venv/Scripts/activate.bat </pre>
Or on Linux/MacOS
<pre>$ source venv/Scripts/activate</pre>
</br>
Install dependencies
<pre>$ pip install -r requirements.txt</pre>
</br>
Run migrations

<pre>$ python manage.py makemigrations</pre>
<pre>$ python manage.py migrate</pre>
</br>
Run server
<pre>$ python manage.py runserver</pre>
</br>
